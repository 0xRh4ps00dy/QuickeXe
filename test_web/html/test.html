Checklist de limpieza DOCX

Antes de subir el archivo

Usa este checklist para preparar tu DOCX y mejorar la conversion al paquete web compatible con eXeLearning.

Checklist obligatorio

  • Aplicar estilos de Word reales para encabezados: Titulo 1, Titulo 2 y Titulo 3.
  • Definir cada bloque tematico principal con un Titulo 1.
  • Mantener una sola columna de lectura en todo el documento.
  • Eliminar cuadros de texto flotantes y elementos superpuestos.
  • Usar listas nativas de Word para vinetas y numeracion.
  • Quitar saltos de linea y parrafos vacios innecesarios.
  • Insertar imagenes en linea con el texto, no ancladas como flotantes.
  • Mantener tablas simples, evitando fusiones complejas de celdas.
  • Reducir encabezados y pies repetitivos que vengan de la conversion PDF.
  • Comprobar que el orden de lectura sea continuo de arriba hacia abajo.

Flujo recomendado

  1. Convertir el PDF a DOCX.
  2. Revisar y limpiar el DOCX con este checklist.
  3. Guardar y subir el DOCX a la app de generacion.
  4. Validar la navegacion del paquete exportado antes de publicarlo.

Limitaciones conocidas

Sin OCR: solo se procesan documentos con texto digital. Si un PDF original tiene maquetacion muy compleja, revisa manualmente la estructura en DOCX antes de subir.
